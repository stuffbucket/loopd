loopd

Export M365 Loop pages to clean GitHub-Flavored Markdown

Quick Start

  1. Copy the export script Click the button below to copy loopd.js to your clipboard
  2. Open a Loop page Navigate to the Loop page you want to export in your browser
  3. Open DevTools Console
    Chrome ⌘⌥J Mac CtrlShiftJ Win
    Firefox ⌘⌥K Mac CtrlShiftK Win
    Safari ⌘⌥C Mac Enable in Develop menu first
    Edge ⌘⌥J Mac CtrlShiftJ Win
  4. Paste and run Paste the script into the console and press Enter

One-click export: Drag this button to your bookmarks bar, then click it on any Loop page:

Export Loop
Setup: Show your bookmarks bar (CtrlShiftB on Windows, ⌘ShiftB on Mac), then drag the button above to it. On any Loop page, click the bookmarklet and wait for the export to complete.

Note: Works on Edge and Chrome. Safari blocks bookmarklets on some sites—use the copy/paste method above instead.

Figma Plugin

Import your Loop exports directly into Figma with the Markdown Importer plugin:

  1. Export the plugin files Run in terminal: ./loopd --export-plugin ~/loopd-figma-plugin
  2. Open Figma desktop app The plugin requires the desktop app, not the web version
  3. Import the plugin Go to Plugins menu → Development → Import plugin from manifest...
  4. Select the manifest file Navigate to ~/loopd-figma-plugin/manifest.json and open it
  5. Run the plugin Plugins → Development → loopd Markdown Importer
