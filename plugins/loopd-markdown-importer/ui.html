Getting Started

Step 1: Export a Loop page
Open your Loop page and paste the loopd.js script in the console to create a .tar file.

Step 2: Import to Figma
Select the .tar file above and click "Import". The plugin will create a structured Figma design with text, images, and styles.

Step 3: Customize
The generated design uses auto-layout frames and text styles for easy editing.

Features
  • Preserves heading hierarchy
  • Embeds images with proper sizing
  • Applies typography and colors
  • Creates auto-layout frames
  • Generates reusable text styles
